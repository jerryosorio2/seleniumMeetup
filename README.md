# Selenium meetup

Selenium meetup examples

## Installation

First clone repository

```bash
git clone https://github.com/jerryosorio2/seleniumMeetup.git
```

Dependencies

```bash
Java 13
Selenium 3.141.59
IntelliJ
```

Execute test

```bash
Just run the test you want to execute (Right click -> run)
```
